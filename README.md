# Wireworld

A wireworld cellular automata simulator in C++.

## Building

### Dependencies

* SFML

```bash
mkdir build
cd build
cmake ..
make
```

## Controls

| Key | Function |
|-|-|
|Space| Pause/Play. |
|R| Reset the grid. |
|Middle Click|Pan the grid|
|Left/Right Click| Change cell state. (empty/wire/head/tail) |
